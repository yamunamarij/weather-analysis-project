# Weather Analysis Project

## Project Overview
This project fetches real-time weather data for selected cities using the OpenWeather API and visualizes the data using Python graphs.

## Features
- Live weather data collection
- Temperature analysis
- Humidity analysis
- Pie chart visualization
- User-selected cities

## Technologies Used
- Python
- Pandas
- Matplotlib
- OpenWeather API

## Graphs Included
- Temperature Bar Chart
- Humidity Bar Chart
- Humidity Pie Chart

## Future Enhancements
- Weather forecasting using Machine Learning
- Air quality monitoring
- Rainfall prediction
- Web application development
